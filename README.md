## Проект CLIcalculator

## 📌 Функциональность

Приложение позволяет:

- вводить параметры прямоугольника (длина `a` и ширина `b`);
    
- вычислять:
    
    - периметр;
        
    - площадь;
        
    - диагональ;
        
    - длину описанной окружности;
        
    - радиус вписанной окружности (только для квадрата);
    
## 🛠 Используемые технологии

## ▶ Запуск проекта

### 1️⃣ Запуск из Eclipse

- Откройте проект в выбранной IDE.
- Найдите основной класс с методом public static void main.
- Нажмите кнопку "Запуск" (обычно зеленая стрелка возле этого класса).
- Программа запустится, и вы сможете вводить данные и получать результаты через консоль IDE.

### 2️⃣ Запуск JAR-версии

В корне проекта присутствует файл:

```
Rectangle-GUI-Calculator.jar
```

Для запуска используется команда:

```
java --module-path "c:\javafx-sdk-25.0.1\lib" --add-modules javafx.controls,javafx.fxml -jar Rectangle-GUI-Calculator.jar
```

Также в репозитории присутствует файл `run.cmd`, позволяющий запустить приложение двойным кликом.
